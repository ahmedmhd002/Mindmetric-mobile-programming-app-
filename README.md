# MindMetric (Mobile EEG App) 🧠📱

MindMetric is a Flutter/Android EEG mobile app that integrates native C++/JNI bindings using the `BrainFlow` library for on-device real-time cognitive workload and emotion tracking.

This repository hosts the mobile application component of the larger EEG emotion recognition system.

## 🌟 Key Features
- **Real-Time Data Acquisition:** Connects directly to EEG hardware (e.g., g.tec Unicorn) via Bluetooth.
- **On-Device Signal Processing:** Uses native C++ libraries for high-performance DSP (filtering, artifact rejection).
- **Live Classification:** Evaluates valence/arousal states in real-time.

## 🛠️ Technologies
`Flutter` `Dart` `C++` `JNI` `BrainFlow` `EEG`
